# TailwindCSS Coffee Style by Hieu Van

## Requirements

- nodejs => v20.13.1
- npm => v10.5.2
- tailwindcss => v3.4.7
- live-server => v1.2.2

## How to use

```bash
$git clone https://github.com/hieuvan/tailwindcss-coffee-style.git
$cd tailwindcss-coffee-style
$npm install
$npm run start:dev
$npm run build:css
```
